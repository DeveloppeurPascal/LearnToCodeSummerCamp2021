# Learn to code Summer camp 2021 - french edition - jour 3

Lors de la troisième journée du bootcamp, Patrick Prémartin a proposé de développer une application mobile de prise de selfies.

Le programme SelfieCam fonctionne sur iOS et Android.

Il utilise la caméra par défaut de l'appareil mobile (smartphone ou tablette) afin d'afficher un flux vidéo à l'écran puis permet de figer l'image en cours pour lui appliquer des effets fournis avec FireMonkey. Une fois l'image satisfaisante, le résultat peut être partagé avec d'autres applications, enregistré sur l'appareil ou publié sur les réseaux sociaux selon les applications installées.

Cette présentation est disponible en rediffusion sur la page https://apprendre-delphi.fr/ltcsc2021-03.php
