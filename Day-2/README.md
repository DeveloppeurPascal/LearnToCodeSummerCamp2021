# Learn to code Summer camp 2021 - french edition - jour 2

3 projets ont été développés par Patrick Prémartin lors de la seconde session du Learn To Code Summer Camp en français sur sa chaîne Twitch.

Cette présentation est disponible en rediffusion sur la page https://apprendre-delphi.fr/ltcsc2021-02.php

## 01-HelloTwitch-FMX

Ce projet FireMonkey est un exemple de base de projet Delphi avec une fenêtre, un bouton et l'affichage d'une boite de dialogue lorsqu'on clique dessus.

Un "Hello World" qui affiche "Hello Twitch".

## 02-Whois

Exemple d'utilisation des composants Indy pour consulter une base de données WHOIS liée aux noms de domaine.

## 03-APIDelphiBooks

Nous avons fini par faire du JSON en utilisant l'API du site https://delphi-books.com qui recense les livres parlant de Pascal, Object Pascal et Delphi.

De façon itérative nous sommes arrivés à un programme avec plusieurs écrans qui synchronisent leurs informations avec le site Internet en affichant la liste des livres, des données sur le livre choisi dont sa fiche (sous forme de page web) et sa couverture.
